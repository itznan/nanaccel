# Writing Custom HLSL Shaders for nanaccel

**nanaccel** allows you to execute custom HLSL Pixel Shaders (`ps_5_0`) directly on video frames at runtime. This document details how to write compatible shaders.

---

## 1. Shader Input Structure

Each pixel shader receives a full-screen triangle generated by the default vertex shader. The texture coordinates (`TEXCOORD0`) map the current frame:

```hlsl
Texture2D shaderTexture : register(t0); // The input video frame (RGBA)
SamplerState SampleType : register(s0); // The standard D3D11 Sampler State

struct VS_OUTPUT {
    float4 Pos : SV_POSITION; // Vertex Position
    float2 Tex : TEXCOORD0;   // Texture UV Coordinates (0.0 to 1.0)
};
```

---

## 2. Basic Shader Template

Your pixel shader must implement the `main` entry point and return a `float4` (representing `RGBA` colors):

```hlsl
float4 main(VS_OUTPUT input) : SV_TARGET {
    // 1. Sample the color from the current video frame texture
    float4 color = shaderTexture.Sample(SampleType, input.Tex);
    
    // 2. Perform custom color/effect logic here
    
    // 3. Return the modified RGBA pixel color
    return color;
}
```

---

## 3. Shader Examples

### Grayscale / Monochrome
Converts each pixel color to grayscale based on standard weight coefficients:
```hlsl
float4 main(VS_OUTPUT input) : SV_TARGET {
    float4 color = shaderTexture.Sample(SampleType, input.Tex);
    float gray = dot(color.rgb, float3(0.299, 0.587, 0.114));
    return float4(gray, gray, gray, color.a);
}
```

### Invert Colors
Inverts all color channels:
```hlsl
float4 main(VS_OUTPUT input) : SV_TARGET {
    float4 color = shaderTexture.Sample(SampleType, input.Tex);
    return float4(1.0 - color.rgb, color.a);
}
```

### Color Channels swap (e.g. Sepia Toning)
Applies warm sepia tone mapping:
```hlsl
float4 main(VS_OUTPUT input) : SV_TARGET {
    float4 color = shaderTexture.Sample(SampleType, input.Tex);
    float3 sepia;
    sepia.r = (color.r * 0.393) + (color.g * 0.769) + (color.b * 0.189);
    sepia.g = (color.r * 0.349) + (color.g * 0.686) + (color.b * 0.168);
    sepia.b = (color.r * 0.272) + (color.g * 0.534) + (color.b * 0.131);
    return float4(sepia, color.a);
}
```
